  • Text
    • Описание - Название товара
    • input
      • Валидация
        • Поле должно быть заполнено (required)
        • Строка должна быть не больше 15 символов (maxlength="15")
  • Radio
    • Описание - Есть в наличии
    • 1й input
      • label - Да
      • value - yes
    • 2й input
      • label - Нет
      • value - no
      • Должен быть выбран по умолчанию (checked)
    • 3й input
      • label - Ожидается
      • value - expected
      • Должен быть отключен (disabled)
  • Color
    • input
      • label - Цвет
      • value - #e66465
  • Сheckbox
    • Описание - Материал
    • 1й input
      • label - Другой
      • value - other
      • Должен быть выбран по умолчанию (checked)
    • 2й input
      • label - Пластмасс
      • value - plastic
    • 3й input
      • label - Металл
      • value - metal
  • File
    • Описание - Фото
    • input
      • Должна быть возможность выбрать несколько фотографий (multiple)
      • Валидация
        • Должна быть возможность выбрать только картинки (accept="image/*")
  • Number
    • Описание - Цена
    • input
      • Валидация
        • Поле должно быть заполнено (required)
        • Число должно быть не меньше 5 (min="5")
        • Число должно быть не больше 5000 (max="5000")
